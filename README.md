# 🏫 Website Landing Page SMK Budi Bakti Ciwidey

Sebuah platform website landing page modern, responsif, dan interaktif yang dirancang khusus untuk mempromosikan profil sekolah, program keahlian (jurusan), serta galeri aktivitas dari SMK Budi Bakti Ciwidey. 

Project ini dideploy secara publik dan dapat diakses melalui tautan berikut:
🔗 **[magical-cendol-050acf.netlify.app](https://magical-cendol-050acf.netlify.app/)**

---

## 🚀 Fitur Utama
* **Responsive Layout:** Tampilan web otomatis menyesuaikan dengan sempurna baik di layar Desktop maupun Mobile (Smartphone).
* **Interactive Modals:** Pop-up detail informasi jurusan (DKV, RPL, Bisnis Digital) dan informasi PPDB yang interaktif menggunakan JavaScript.
* **Modern UI Components:** Desain bernuansa gelap (*Dark Mode Theme*) dengan aksen hiasan efek lingkaran berpendar (*glow effects*) yang estetis.
* **Navigation Menu (Mobile Friendly):** Menu navigasi ringkas (*hamburger menu*) yang dinamis untuk perangkat seluler.

## 🛠️ Teknologi yang Digunakan
* **HTML5** - Sebagai struktur dasar semantik halaman web.
* **Tailwind CSS (via CDN)** - Framework CSS berbasis *utility-first* untuk styling cepat dan modern.
* **JavaScript (Vanilla)** - Untuk logika interaksi komponen modal dan menu responsif.
* **Font Awesome v6.4.0** - Toolkit ikon web untuk mempercantik elemen antarmuka.

## 💻 Cara Menjalankan Project Secara Lokal

Jika ingin menjalankan project ini di komputer lokal Anda, ikuti langkah berikut:

1. **Clone atau Unduh Project:**
   Pastikan seluruh file (`index.html` beserta aset gambar seperti `LOGO BBC.jpg`, `bbc hero.jpeg`, dll.) berada dalam satu folder yang sama.

2. **Buka dengan Text Editor:**
   Buka folder proyek menggunakan editor kode pilihan Anda (disarankan menggunakan *Visual Studio Code*).

3. **Jalankan via Local Server:**
   * Jika menggunakan VS Code, klik kanan pada file `index.html` dan pilih **Open with Live Server**.
   * Atau, cukup klik dua kali pada file `index.html` untuk membukanya langsung di browser pilihan Anda.
